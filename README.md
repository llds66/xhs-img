# xhs-img-vue

使用 `Vue + Markdown` 生成小红书风格图文（黑底网格、标题段落、强调色、页码），并在网页中导出 PNG。

## 1. 安装依赖

```bash
npm install
```

## 2. 本地预览

```bash
npm run dev
```

打开后可在左侧编辑内容，右侧实时查看分页预览。

## 3. 网页导出 PNG

- 导出当前页：点击「导出当前页」
- 导出全部页（ZIP）：点击「导出全部（ZIP）」

## 4. Markdown 规则

- `#`：封面主标题
- `##`：页内小标题
- 普通段落：正文
- `**文字**`：强调高亮
- `---`：手动换页
